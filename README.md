##  Features

- **Modern UI/UX**: Built with **NextUI** and **Tailwind CSS 4** for a sleek, responsive, and accessible design.
- **Dynamic Animations**: Seamless transitions and micro-interactions powered by **Framer Motion**.
- **Interactive Backgrounds**: Immersive 3D/Geometric visuals using **Three.js** (@react-three/fiber).
- **Comprehensive Showcase**:
  - **Expertise Area**: Highlighting specialized skills in Web3, Frontend, and Backend.
  - **Project Gallery**: A curated list of live projects with GitHub and Demo links.
  - **Infinite Scroll Carousel**: An engaging way to browse through featured work.
- **Functional Contact Form**: Integrated with **EmailJS** for direct client communication.
- **Theme Support**: Built-in dark mode optimization for a premium feel.

##  Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm / pnpm / yarn

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Vheek10/Portfolio.git
    cd Portfolio/portfolio
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    # or
    pnpm install
    ```

3.  **Environment Variables**:
    Create a `.env.local` file in the root directory and add your EmailJS credentials:
    ```env
    NEXT_PUBLIC_EMAILJS_PUBLIC_KEY=your_public_key
    NEXT_PUBLIC_EMAILJS_SERVICE_ID=your_service_id
    NEXT_PUBLIC_EMAILJS_TEMPLATE_ID=your_template_id
    ```

4.  **Run the development server**:
    ```bash
    npm run dev
    ```

5.  **Open in Browser**:
    Navigate to [http://localhost:3000](http://localhost:3000) to see the result.
